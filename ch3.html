Statistical Test: The F Test

§1

為什麼需要統計檢定？

第二章學會了計算相關係數 r。但問題來了：

🤔 關鍵問題
假設我從 6 位受試者算出 r = −0.5。這個相關真的存在嗎？還是純粹是抽樣巧合？
如果重新抽 6 個人，會不會得到 r = +0.2、或 r = 0？

這正是推論統計（Inferential Statistics）要解決的問題。我們不只想知道樣本的相關，更想知道母群體的真實相關。

統計檢定的核心問題
樣本 r → 母群體 ρ = 0 嗎？
我們用 F 比值來判斷：
F = r² / (1−r²) × (S−2)
其中 S 是觀察值數量，(S−2) 稱為「誤差自由度」
📊

樣本 vs 母群體

樣本（Sample）：實際測量的觀察值
母群體（Population）：要推論的整體

樣本叫「statistics」，母群體真值叫「parameters」

🎲

機率推論

無法 100% 確定，只能用機率：「如果母群體相關為 0，有多少機率會抽到像我這樣的樣本？」

§2

虛無假設 vs 對立假設

假設檢定的核心是兩個對立的假設：

∅

虛無假設 H₀
(Null Hypothesis)

陳述：母群體兩變數沒有線性關係（ρ = 0）

性質：「精確」假設（明確指定 ρ=0）

∃

對立假設 H₁
(Alternative Hypothesis)

陳述：母群體兩變數有線性關係（ρ ≠ 0）

性質：「不精確」假設（沒指定確切數值，涵蓋無限多種可能）

決策邏輯
如果 H₀ 看起來不太可能 → 拒絕 H₀，接受 H₁
我們無法直接「證明」H₁，只能「反證」H₀。
如果樣本結果在 H₀ 為真時機率很小，就有信心拒絕 H₀。
但反過來不行：永遠不能「接受」H₀，只能「無法拒絕」（暫緩判斷）。
⚠️ 重要：永遠不能「接受 H₀」
「無法拒絕 H₀」 ≠ 「H₀ 為真」。可能是樣本不夠、效果太小、檢定力不足。正確說法是「暫緩判斷（Suspend Judgment）」。
§3

顯著水準 α：多不可能才算「不可能」？

判斷 H₀「不太可能」需要一個明確的機率門檻，稱為「顯著水準」，以希臘字母 α 表示。

傳統的「儀式性」顯著水準
α = .05　或　α = .01
α = .05：1/20，意思是「如果 H₀ 為真，結果發生機率小於 5%」
α = .01：1/100，更嚴格的標準

這兩個值來自 Fisher 在 20 世紀初的偏好（加上 10 進位的便利性），並非神聖不可改變。
💡 歷史背景：早年沒電腦，計算機率非常困難。Fisher 編製機率表時只能選幾個 α 值。.05 對應 1/20，.01 對應 1/100，都是「10 進位整數倍」——於是成了標準。
§4

兩種錯誤：Type I 與 Type II

無論 α 設多嚴格，都不可能完全避免錯誤。統計檢定有兩種錯誤類型：

實際 H₀ 為真
（無真實關係）		實際 H₀ 為假
（有真實關係）
不拒絕 H₀✅ 正確不拒絕
機率 = 1−α		❌ Type II 錯誤
（漏失 Miss）
機率 = β
拒絕 H₀❌ Type I 錯誤
（假警報）
機率 = α		✅ 正確偵測
機率 = 1−β = 檢定力
🚨

Type I 錯誤（假警報）

實際沒效果，我們卻說有

機率 = α（直接控制）
嚴格的 α（如 .01）會減少 Type I 錯誤

😴

Type II 錯誤（漏失）

實際有效果，我們卻沒發現

機率 = β（無法精確計算）
寬鬆 α（如 .10）會減少 Type II 錯誤

α 與 β 的取捨
α ↓ ⇄ β ↑
兩種錯誤反向變動。要降低 β（增加檢定力），可以：(1) 提高 α、(2) 增加樣本數、(3) 減少實驗誤差。
🎯 檢定力 = 1 − β
檢定力（Statistical Power）是「當效果真實存在時，我們能成功偵測到的機率」。心理學界一般建議檢定力應達 .80 以上。
§5

Fisher 的 F 分配

用一個小例子說明：我們從 6 位受試者算出 r = −0.5。如何判斷在 H₀ 為真時的機率？

F 比值公式
F = r² / (1−r²) × (S−2)
本例：F = (.25) / (1−.25) × (6−2) = (.25/.75) × 4 = 1.33
ν₁ = 1（分子自由度），ν₂ = S − 2 = 4（分母自由度）

當 H₀ 為真時，F 值會服從一個特定的分配——Fisher's F 分配。它的形狀由兩個自由度（ν₁, ν₂）決定。

📊 Fisher F 分配（ν₁=1, ν₂=4）：H₀ 為真時 F 值的機率分布
紅線 = α=.05 臨界值（F=7.71）　橘線 = α=.01 臨界值（F=21.20）
🎲

Monte-Carlo 法

用電腦實際模擬數千個 H₀ 為真的樣本，觀察 F 的分布。直觀但需大量計算。

📐

理論 F 分配

Fisher 數學推導出的理論分配。前提：母群體呈常態分配。可查表得到臨界值。

🔄

排列檢定

計算所有可能的資料排列（如 6!=720 種），看其中多少 F 超過觀察值。最精確。

§6

臨界值與拒絕區

臨界值（Fcritical）把 F 分配劃分成兩個區域：

🚫

拒絕區（Rejection Region）

F ≥ Fcritical 的範圍

機率 ≤ α → 拒絕 H₀，接受 H₁

看到顯著結果！

⏸️

暫緩判斷區

F < Fcritical 的範圍

機率 > α → 無法拒絕 H₀

結果不顯著（但不代表 H₀ 為真）

📋 F 表查表範例

ν₂ \ ν₁1234
47.71 / 21.206.94 / 18.006.59 / 16.696.39 / 15.98
184.41 / 8.283.55 / 6.013.16 / 5.092.93 / 4.58
204.35 / 8.103.49 / 5.853.10 / 4.942.87 / 4.43
404.08 / 7.313.23 / 5.182.84 / 4.312.61 / 3.83
每格上方為 α=.05 臨界值，下方為 α=.01 臨界值
📖 例題：S=6 → ν₂=4。算出 F=1.33。查表：Fcritical(α=.05)=7.71。
因為 1.33 < 7.71，無法拒絕 H₀。結論：暫緩判斷，r=−0.5 可能是巧合。
§7

統計檢定六步驟

1

提出假設 H₀ 與 H₁

H₀：母群體無線性關係（ρ=0）
H₁：母群體有線性關係（ρ≠0）

2

選擇統計量

對相關分析：使用 F 比值（也可用 r² 或 t）

3

決定顯著水準 α

傳統：α=.05 或 .01
權衡：嚴格 α 減少 Type I，但增加 Type II

4

找出 H₀ 下的抽樣分布

F 分配，自由度為 ν₁=1，ν₂=S−2

5

查表得 Fcritical

根據 α 和自由度查 F 表，畫出拒絕區與暫緩判斷區

6

計算 F 並做決策

F ≥ Fcritical：拒絕 H₀（顯著）
F < Fcritical：暫緩判斷（不顯著）

§8

「不是零」還不夠！效果量的重要性

拒絕 H₀ 只說「相關不是 0」，但無法告訴我們相關有多大。現代統計學主張：除了顯著性，更要報告效果量與信賴區間。

🎯 樣本 r 會「高估」母群體相關！
樣本 r 是「偏誤估計量」，系統性大於母群體真值。樣本越小，高估越嚴重。
解決方法：使用「校正後 r²」（Shrunken / Adjusted r）。
校正公式 1（最常用）
r̃² = 1 − (1−r²) × (S−1)/(S−2)
本例：r=−0.5, S=6 → r²=.25
r̃² = 1 − (1−.25)×(5/4) = 1 − .9375 = 0.06
r̃ = −√0.06 = −0.24（要還原原本的負號）
💡 校正的啟示：原本看似有意義的 r=−0.5，校正後其實只有 −0.24（或更小）。這就是為什麼小樣本研究的結果需要重複驗證才能信任。
§9

信賴區間：比顯著性更有資訊

單純說「顯著/不顯著」太簡化。信賴區間（CI）給出母群體 r 的「可能範圍」，資訊量大得多。美國心理學會（APA）建議優先使用信賴區間。

🔢 Fisher's Z 轉換

因為 r 的抽樣分布不對稱，需先把 r 轉成 Z（這個 Z 的抽樣分布近似常態）：

r 轉 Z（Fisher's Z transform）
Z = ½ × [ln(1+r) − ln(1−r)]
本例：r=−0.5 → Z = ½ × [ln(0.5) − ln(1.5)] = −0.5493

📏 計算 95% 信賴區間

1

選 α，查 Z 表

α=.05 → Zα=1.960
α=.01 → Zα=2.575

2

計算 Q（標準誤）

Q = Zα × √(1/(S−3))
本例：Q = 1.960 × √(1/3) = 1.1316

3

計算 Z 上下限

Z 下限 = −0.5493 − 1.1316 = −1.6809
Z 上限 = −0.5493 + 1.1316 = +0.5823

4

把 Z 轉回 r

r 下限 = −0.93　　r 上限 = +0.52
95% 信賴區間為 [−0.93, +0.52]

📊 解讀：母群體真實相關 r 可能介於 −0.93 到 +0.52 之間——範圍非常寬！因為 0 在這個區間內，所以無法拒絕 H₀。

但 CI 還告訴我們：真實效果可能非常強（負相關），也可能完全不存在或微弱正相關——這些是 p 值給不了的。

🎲 Bootstrap：現代替代方案

Bootstrap（自助法，Efron, 1979）是現代強大的方法。原理：把樣本當「迷你母群體」，從中重複抽樣（可重複抽取）千百次，得到統計量的經驗分布，再從中讀百分位數作為 CI。

本例 1000 次 bootstrap 後的 95% CI 為 [−.95, +.41]，與 Fisher 法的 [−.93, +.52] 相當吻合。Bootstrap 不依賴常態分配假設，是現代統計學最重要的進展之一。
