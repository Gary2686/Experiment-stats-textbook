§1

什麼是線性迴歸？

線性迴歸（Linear Regression）是一種用於分析實驗資料的統計技術。它的核心目標是：用自變數（X）來預測依變數（Y）。

使用線性迴歸有個前提：自變數和依變數都必須是量化（連續）變數。這是它與其他分析方法的重要區別。

📌

自變數（X）

由研究者操控的變數，也稱為預測變數（Predictor）。在迴歸分析中，X 繪製在橫軸（水平軸）。

🎯

依變數（Y）

由研究者測量的結果變數，也稱為效標變數（Criterion）。Y 繪製在縱軸（垂直軸）。

🔮

預測值（Ŷ）

由迴歸方程式計算出的 Y 估計值，讀作「Y hat」（Y 帽）。它與實際觀察值 Y 是不同的概念。

💡 迴歸 vs. 相關：相關分析（第三章）告訴我們兩個變數之間的關係強度；迴歸分析進一步讓我們用一個變數去預測另一個變數的數值。迴歸是相關的延伸與應用。
§2

直線方程式：迴歸的基礎

迴歸分析的核心是一條直線。任何一條直線都可以用以下方程式來描述：

迴歸方程式 Regression Equation
Ŷ = a + b × X
Ŷ（Y hat）= 依變數的預測值
a = 截距（Intercept）：當 X = 0 時，Ŷ 的值（直線與 Y 軸的交叉點）
b = 斜率（Slope）：X 每增加 1 單位，Ŷ 改變的量（「上升除以水平移動」）
X = 自變數的值
🎛️ 互動演示：調整截距與斜率，觀察直線如何變化
目前方程式：Ŷ = 1 + 2X

斜率 b 的計算方式：取兩個 X 值及其對應的 Ŷ 值，計算 ΔŶ ÷ ΔX（Ŷ 的變化量除以 X 的變化量），這就是「上升除以水平移動」（rise over run）的概念。

📌 完美直線 vs. 現實資料：在真實的實驗資料中，資料點不可能完美地落在一條線上。因此，我們需要找到一條「最佳擬合線」來盡可能描述資料的趨勢。
§3

迴歸線是「最佳擬合線」

當資料點不能完美落在直線上時，我們引入兩個關鍵概念：誤差（Error）和殘差（Residual）。

對於每個觀察值，實際值 Y 和預測值 Ŷ 之間的差距稱為殘差：

殘差 Residual
殘差 = Y − Ŷ
殘差代表預測的「誤差」——我們試圖用迴歸線預測 Y 之後，剩餘的、無法解釋的部分。
殘差可正可負，平均值為零。

最小平方法（Ordinary Least Squares, OLS）：迴歸線是使「所有殘差的平方和」達到最小的那條線：

最小化目標函數
min Σ(Y − Ŷ)² = Σ[Y − (a + bX)]²
我們要找到使上式達到最小值的 a（截距）和 b（斜率）。
選擇「平方」而非「絕對值」的原因：使用微積分求導時，平方函數比絕對值函數更容易處理。
🧮

斜率 b 的公式

b = SCPXY / SSX = (σ̂Y / σ̂X) × rX·Y

SCPXY = Σ(X−MX)(Y−MY)：XY 的交叉乘積和
SSX = Σ(X−MX)²：X 的平方和

📍

截距 a 的公式

a = MY − b × MX

截距由依變數平均值（MY）和自變數平均值（MX）決定。迴歸線必定通過點 (MX, MY)。

🔑 重要性質：迴歸線必定通過資料的均值點 (MX, MY)。預測值 Ŷ 的均值等於觀察值 Y 的均值（MŶ = MY）。
§4

實例：反應時間與記憶組大小

本章以 Sternberg（1969）實驗的複製研究為例，說明迴歸分析的應用。

🧪 實驗設計

研究假設：受試者在短期記憶中序列掃描字母，每個字母所需掃描時間相同。因此，記憶組大小（字母數）應與反應時間呈線性正相關。

自變數（X）：記憶組大小（1、3、5、7個字母）

依變數（Y）：「是」答案的平均反應時間（毫秒，ms）

受試者：20位（每組5位），各組只接受一種記憶組大小的測試

受試者 X（組大小） Y（反應時間ms） x = X−MX x² y = Y−MY xy
Σ（總和） 80 11,200 0 100 (SSX) 0 4,000 (SCP)
1

計算基本統計量

MX = 80/20 = 4  ｜  MY = 11,200/20 = 560
SSX = Σ(X−MX)² = 100  ｜  SCPXY = Σ(X−MX)(Y−MY) = 4,000

2

計算斜率 b

b = SCPXY / SSX = 4,000 / 100 = 40
意義：記憶組每增加1個字母，反應時間增加 40 ms（即掃描一個字母所需時間）

3

計算截距 a

a = MY − b × MX = 560 − 40 × 4 = 560 − 160 = 400
意義：處理探測字母和產生回答所需的基礎時間為 400 ms

4

寫出迴歸方程式

預測反應時間 = 400 + 40 × 記憶組大小
Ŷ = 400 + 40X

📈 散佈圖與迴歸線（Sternberg 實驗資料）
藍點 = 實際觀察值  ｜  紅線 = 迴歸線 Ŷ = 400 + 40X  ｜  虛線 = 殘差
§5

如何評估預測品質？

我們知道迴歸方程式給出了最佳預測，但這個預測究竟有多好？評估方式：計算預測值 Ŷ 與實際值 Y 之間的相關係數 rŶ·Y。

決定係數 Coefficient of Determination
r²X·Y = SSregression / SStotal
r² 代表 X 能解釋 Y 變異的比例。
本例：r = .9950，故 r² = .9900，表示 99% 的 Y 變異可由 X 解釋。

評估預測是否顯著優於純粹猜測（即使用 MY 作為預測），需要計算 F 比值：

F 比值（方法一）
F = r² / (1−r²) × (S−2)
本例：F = (.9950²) / (1−.9950²) × (20−2) = 1,800.00
ν₁ = 1，ν₂ = S−2 = 18
查 F 表：α=.05 臨界值 = 4.41，α=.01 臨界值 = 8.28
F = 1,800 >> 兩個臨界值 → 拒絕 H₀，記憶組大小確實線性預測反應時間
✅ 結論：迴歸分析顯示，記憶組大小（X）能夠顯著線性預測反應時間（Y），F(1,18) = 1800, p < .01，r² = .99，預測效果極佳。
§6

平方和的分割（變異來源分析）

這一節引入一個重要觀念：Y 的總變異（SStotal）可以分割為兩個部分——

變異分割 Partitioning of Variance
SStotal = SSregression + SSresidual
SStotal = Σ(Y−MY)² → Y 的總變異（共 161,600）
SSregression = Σ(Ŷ−MY)² → 由 X（迴歸）解釋的變異（共 160,000）
SSresidual = Σ(Y−Ŷ)² → 無法解釋的殘差變異（共 1,600）

📊 變異來源視覺化（本例數值）

SStotal 總變異161,600
SS迴歸 (99%)
SSregression = 160,000（99.0%）由 X 解釋
SSresidual = 1,600（1.0%）無法解釋

迴歸分析摘要表（ANOVA Table）

變異來源 SS（平方和） df（自由度） MS（均方） F 比值
迴歸 (Regression) 160,000 1 160,000 1,800.00
p < .01
殘差 (Residual) 1,600 18 88.89
總計 (Total) 161,600 19 — —
📐

自由度說明

dfregression = 1：畫一條線只需要固定兩個點，均值點已固定，只需再一個點（自由度1）。

dfresidual = S−2 = 18：殘差受到兩個限制（均值=0，且與X無關），因此損失2個自由度。

⚖️

F 比值的意義

F = MSregression / MSresidual

F > 1 表示迴歸方差大於殘差方差，即迴歸線比均值更能預測 Y。F 值越大，預測效果越好。
