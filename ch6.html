第六章 · Chapter 6

非正交多元迴歸
Non-orthogonal Multiple Regression

當兩個自變數彼此「有相關」時，事情變得棘手——但也才是真實研究的樣貌。本章學習如何處理「共享變異」的問題。

§1

為什麼需要這一章？

第五章告訴我們：當兩個自變數正交時，多元迴歸超簡單。但現實中，研究者不總能讓自變數正交。

🚨 什麼時候不能正交？
當研究中有「標籤變數（tag variable）」時——也就是研究者無法操弄、受試者本身就帶著的特徵（第 1 章提過）。例如：

· 年齡、性別、智商、收入、社經地位、種族⋯⋯
· 這些變數研究者只能「測量」而非「分派」
· 它們之間通常彼此相關，無法設計成正交
🎯 本章核心問題
當 X 和 T 彼此有相關時，它們對 Y 的解釋會「重疊」。
那麼，每個自變數真正獨有的貢獻是多少？

解答：使用「半偏相關係數（semi-partial correlation）」——專門用來分離「共享變異」與「獨特貢獻」的利器。
😊

正交（第5章）

X 和 T 的相關 = 0

· 計算簡單
· 解釋直觀
· 各變數效果可加

r²Y·X + r²Y·T = R²

😰

非正交（本章）

X 和 T 的相關 ≠ 0

· 計算複雜
· 解釋微妙
· 變異有重疊

r²Y·X + r²Y·T ≠ R²

§2

實例：兒童記憶廣度（Hulme et al., 1984）

🧒 研究背景與問題

Hulme 等人（1984）想了解：什麼因素決定兒童的短期記憶廣度？

他們找來 4、7、10 歲的兒童，測量：

· X = 年齡（歲）
· T = 語速（每秒能讀出幾個字）
· Y = 記憶廣度（10 次測試中正確回憶的總字數）

研究假設：作者認為「語速」才是決定記憶的真正原因，而「年齡」的效果只是透過語速顯現出來。換句話說：年紀大不一定記得多，是因為「年紀大的小孩通常講話比較快」，才看起來記得多。

⚠️ 為什麼這個研究「天生」就是非正交？
因為年齡和語速本來就有相關——年紀越大的小孩，通常語速越快。研究者無法強迫某些 4 歲小孩講超快，或讓 10 歲小孩講超慢。這就是「標籤變數」的宿命。

§3

計算迴歸平面（公式完整版）

第 5 章的正交簡化公式在這裡不能用了！必須回到完整版：

完整版斜率 b（X 方向）
b = (SST·SCPXY − SCPXT·SCPTY) / (SSX·SST − SCPXT²)
跟第 5 章公式完全相同，只是這次 SCPXT ≠ 0 不能被消掉。
這個公式的本質：「把 T 對 X 的影響先扣掉，再算 X 對 Y 的純效果」。

🧮 套入範例數值

統計量意義數值
SSXX 的平方和36
SSTT 的平方和16
SSYY 的平方和1,846.83
SCPXYX 與 Y 的交叉乘積和207
SCPTYT 與 Y 的交叉乘積和170
SCPXTX 與 T 的交叉乘積和18 ← 不是 0！
1

計算 b（年齡的斜率）

b = (16 × 207 − 18 × 170) / (36 × 16 − 18²)
  = (3,312 − 3,060) / (576 − 324)
  = 252 / 252 = 1.00

每多 1 歲，多記 1 個字（控制語速後）

2

計算 c（語速的斜率）

c = (36 × 170 − 18 × 207) / (36 × 16 − 18²)
  = (6,120 − 3,726) / 252
  = 2,394 / 252 = 9.50

語速每加快 1 字/秒，多記 9.5 個字（控制年齡後）

3

計算 a（截距）

a = MY − b·MX − c·MT
  = 37.17 − 1 × 7 − 9.5 × 3
  = 37.17 − 7 − 28.5 = 1.67

4

寫出迴歸方程式

Ŷ = 1.67 + 1.00X + 9.50T

意思：記憶字數 = 1.67 + 1×（年齡）+ 9.5×（語速）

💡 初步解讀：兩個斜率都是正的，符合預期：年齡和語速都對記憶有正面影響。但誰才是真正的關鍵？這要看後面的「半偏相關」分析。
§4

整體 R²：兩變數合起來解釋多少？

多元決定係數
R²Y·XT = SCPŶY² / (SSŶ · SSY)
本例：R² = 1,822² / (1,822 × 1,846.83) = 0.9866
🎯 解讀：年齡和語速合起來解釋了 98.66% 的記憶變異！剩餘只有 1.34% 是無法解釋的誤差。模型很強。

F 檢定：模型顯著嗎？

整體 F 比值
F = R²/(1−R²) × (S−K−1)/K
S = 6, K = 2 → F = 0.9866/0.0134 × 3/2 = 73.63 × 1.5 = 110.50
查表 F臨界(2, 3, α=.05) = 9.55
110.50 ≫ 9.55 → 拒絕 H₀，整體模型顯著
§5

麻煩來了：加總超過 1！

分別計算 X 和 T 與 Y 的相關平方，看看會發生什麼：

單獨相關計算數值
r²Y·X（年齡單獨解釋）207² / (1,846.83 × 36)0.6445 (64.45%)
r²Y·T（語速單獨解釋）170² / (1,846.83 × 16)0.9780 (97.80%)
兩者相加0.6445 + 0.97801.6225 ❌
整體 R²—0.9866
🚨 很怪的事情！
r²Y·X + r²Y·T = 1.6225 > R² = 0.9866

變異總和居然 超過 100%！這顯然不合理——任何「比例」都不可能超過 1。

原因是：X 和 T 之間有相關（rXT = 0.75），它們各自對 Y 的解釋有重疊。當我們簡單把它們加起來，重疊的部分被算了兩次。
📐 對比第 5 章
第 5 章正交時：0.6489 + 0.2884 = 0.9373 ≈ R² ✓
第 6 章非正交時：0.6445 + 0.9780 = 1.6225 ≫ R² ❌

關鍵差異：正交時，X 和 T 對 Y 的解釋「無重疊」，可以乾淨切割；非正交時，它們的解釋「重疊」，必須用新工具來分離。
§6

半偏相關：本章的英雄

為了解決「重疊計算兩次」的問題，我們需要一個新概念：

半偏相關（Semi-partial Correlation）
r²Y·X|T　讀作「Y 與 X，扣除 T 影響後的相關平方」
這代表「X 對 Y 的獨特貢獻」——也就是 T 解釋不了、只有 X 能解釋的部分。
別名：「part correlation」
為什麼叫「半」偏？因為我們只從 X 中扣除 T 的影響，沒有從 Y 中也扣除。
🎬 用劇場比喻：
想像 X（年齡）和 T（語速）是兩位演員，Y（記憶）是票房。
· 一般相關 r²Y·X：年齡「掛名」的票房（但其實有語速幫忙）
· 半偏相關 r²Y·X|T：把語速的功勞扣掉後，年齡純粹能帶來的票房

🔧 怎麼算半偏相關？三步驟

1

把 X 用 T 來預測

先計算：「如果只用語速 T，能預測年齡 X 嗎？」

X̂T = aX·T + bX·T·T
bX·T = SCPXT / SST = 18 / 16 = 1.125
aX·T = MX − 1.125 × MT = 7 − 1.125 × 3 = 3.625

用語速預測年齡的方程式：X̂T = 3.625 + 1.125 × T

2

計算「殘差」：X 中扣除 T 解釋後的部分

eX·T = X − X̂T　（X 的實際值減去用 T 預測的值）

這個殘差代表「X 中無法被 T 解釋的部分」，也就是 X 的「獨特成分」。

關鍵性質：殘差 eX·T 與 T 完全不相關（這是迴歸的數學性質）

3

計算 Y 與這個「乾淨的 X」的相關

r²Y·X|T = SCPY·eX·T² / (SSY · SSeX·T)
本例：r²Y·X|T = 15.75² / (1,846.83 × 15.75) = 0.0085

同樣地，T 對 Y 的獨特貢獻：r²Y·T|X = 0.3421

💥 結果太震撼！
· 年齡「看起來」解釋了 64.45% 的記憶變異
· 但扣掉語速的影響後，年齡真正獨自貢獻的只有 0.85%！

這完美驗證了原作者的假設：年齡之所以看起來有影響，是因為它和語速相關。扣掉語速後，年齡本身幾乎沒影響。

變異分解：用 Venn 圖看懂

把 Y 的整個變異拆解成四個部分：

🎨 變異分解圖：四個區塊的故事
左：自變數 X 與 T 的 Venn 圖（重疊區為共享變異）  |  右：Y 的變異分解
變異來源符號比例意義
X 獨特貢獻r²Y·X|T0.85%扣掉 T 後，X 還能解釋的部分
T 獨特貢獻r²Y·T|X34.21%扣掉 X 後，T 還能解釋的部分
X 與 T 共享R² − (兩者半偏和)63.60%兩變數共同解釋（無法歸功於誰）
無法解釋1 − R²1.34%誤差項
合計—100.00%—
變異分解的公式（重要）
Y 的變異 = X 獨特 + T 獨特 + 共享 + 殘差
1 = r²Y·X|T + r²Y·T|X + 共享 + (1 − R²)
本例：1 = 0.0085 + 0.3421 + 0.6360 + 0.0134 ✓

共享變異 = R² − (r²Y·X|T + r²Y·T|X)
        = 0.9866 − 0.3506 = 0.6360
🧐 看到了嗎？共享變異佔了 63.60%！
這意味著 X 和 T 共同解釋的部分非常大。也就是說：「年齡 ∩ 語速」這個重疊部分，是解釋記憶力的主要力量。

這也是為什麼簡單的 r²Y·X 會「灌水」——它包含了 X 獨特的 0.85% 加上 共享的 63.60% = 約 64.45%。
§8

半偏相關的替代公式

不想跑「先預測 X、算殘差、再算相關」三大步嗎？有更快的公式：

替代公式 1：從整體 R² 減
r²Y·X|T = R²Y·XT − r²Y·T
意思：X 的獨特貢獻 = 「兩個變數合起來的解釋」− 「T 自己能解釋的」
本例：r²Y·X|T = 0.9866 − 0.9780 = 0.0086（與前面 0.0085 一致）
替代公式 2：只用三組相關係數
r²Y·X|T = (rY·X − rY·T·rX·T)² / (1 − r²X·T)
本例：rX·T = 0.75, rY·X = 0.8028, rY·T = 0.989
r²Y·X|T = (0.8028 − 0.989×0.75)² / (1 − 0.75²)
         = (0.8028 − 0.7418)² / 0.4375 ≈ 0.0085
💡 實用建議：手算時用替代公式 1 最快——只要先算出 R² 和兩個單獨相關 r²，扣一下就完成。電腦會自動處理。
§9

F 檢定：獨特貢獻顯著嗎？

跟前面類似，半偏相關也可以做 F 檢定，判斷該變數的「獨特貢獻」是否顯著：

半偏相關的 F 比值
FY·X|T = r²Y·X|T / (1 − R²Y·XT) × (S − K − 1)
ν1 = 1，ν2 = S − K − 1
注意：分母是 (1 − R²Y·XT)，不是 (1 − r²Y·X|T)！這代表用整體模型的殘差作為基準。
檢定計算F 值查表臨界值
(α=.05, df=1, 3)		結論
年齡的獨特效果0.0085/(1−0.9866)×31.9110.13❌ 不顯著
語速的獨特效果0.3421/(1−0.9866)×376.5910.13✅ 顯著
🎯 戲劇性的結論
研究最終發現：

· 語速：F = 76.59，p < .05，真的能獨立預測記憶
· 年齡：F = 1.91，p > .05，無法獨立預測記憶

這完全支持了 Hulme 等人的原始假設：「年齡的影響是假象——它只是透過語速來影響記憶。如果兩個小孩語速相同，他們的記憶差不多，年齡不重要。」
🔬 這就是「控制變數」的力量！
半偏相關就是統計學版的「控制變數」。它讓我們在不能用實驗操弄的情況下，仍能分離出每個變數的純效果。

這個技巧在心理學、社會學、流行病學的觀察研究中極為重要。
§10

陷阱：多重共線性（Multicollinearity）

非正交多元迴歸有一個特別危險的陷阱：

🚨 什麼是多重共線性？
當某個自變數「幾乎可以被其他自變數完全預測」時，就稱為多重共線性。

極端例子：如果 X 和 T 的相關 rX·T = 1（完美相關），半偏相關公式會變成：
r²Y·X|T = (...)² / (1 − 1²) = (...)² / 0 ← 除以 0！
為什麼是死亡陷阱
當 r²X·T → 1，分母 (1 − r²X·T) → 0
即使不到完美相關，只要「非常相關」（如 r > 0.9），分母就會變得非常小。
微小的數值波動會被放大成劇烈變化，結果不可靠。
🚧

後果

· 斜率估計極不穩定
· 標準誤暴增
· F 檢定可能拒絕不該拒絕的
· 結論完全不可信

🔍

怎麼診斷？

· 檢查兩兩變數相關矩陣
· 計算 VIF（變異膨脹因子）
· VIF > 10 通常代表有問題
· 看條件指標（CI）

🛠️

怎麼處理？

· 移除高度相關的變數
· 合併成複合變數
· 使用主成分分析
· 使用 ridge regression

💡 實務建議：在跑多元迴歸之前，先檢查自變數兩兩之間的相關係數。如果發現有 |r| > 0.8，就要警覺，可能有共線性問題。
§11

加碼：偏相關（Partial Correlation）

還有一個跟半偏相關「很像但不一樣」的概念，順便認識一下：

✂️

半偏相關 r²Y·X|T

只從 X 扣掉 T

· 把 X 中由 T 解釋的部分扣掉

· Y 保持原樣

「半」= 只處理一半（只處理自變數）

✂️✂️

偏相關 r²(Y·X)|T

從 X 和 Y 都扣掉 T

· 把 X 中由 T 解釋的部分扣掉

· 也把 Y 中由 T 解釋的部分扣掉

「全偏」= 兩邊都處理

偏相關公式
r²(Y·X)|T = (rY·X − rY·T·rX·T)² / [(1 − r²Y·T)(1 − r²X·T)]
跟半偏公式比，只差分母多了一個 (1 − r²Y·T) 項
本例：r²(Y·X)|T = (0.8028 − 0.989×0.75)² / [(1 − 0.989²)(1 − 0.75²)] ≈ 0.3894
🤔 什麼時候用哪個？

半偏相關：當你想問「X 對整個 Y 的獨特貢獻是多少？」(更常用於迴歸分析)

偏相關：當你想問「X 與 Y 之間的純粹關係是什麼？」(扣除 T 的共同影響後，X 與 Y 的『真實』相關)

心理學研究中，半偏相關（即「control for」）使用更頻繁。
§K

關鍵公式整理

完整斜率 b
b = (SS_T·SCP_XY − SCP_XT·SCP_TY) / (SS_X·SS_T − SCP_XT²)
完整斜率 c
c = (SS_X·SCP_TY − SCP_XT·SCP_XY) / (SS_X·SS_T − SCP_XT²)
半偏相關（殘差法）
r²_Y·X|T = SCP_Y·eXT² / (SS_Y × SS_eXT)
半偏相關（替代1：增量法）
r²_Y·X|T = R²_Y·XT − r²_Y·T
最常用：直接從 R² 減
半偏相關（替代2：r 公式）
r²_Y·X|T = (r_Y·X − r_Y·T·r_X·T)² / (1 − r²_X·T)
偏相關
r²_(Y·X)|T = (r_Y·X − r_Y·T·r_X·T)² / [(1 − r²_Y·T)(1 − r²_X·T)]
變異分解
1 = r²_Y·X|T + r²_Y·T|X + 共享 + 殘差
共享變異
共享 = R² − (r²_Y·X|T + r²_Y·T|X)
F 檢定（半偏）
F_Y·X|T = r²_Y·X|T / (1 − R²_Y·XT) × (S − K − 1)
ν₁=1, ν₂=S−K−1
3 個 IV 的情況
r²_Y·X|TQ = R²_Y·XTQ − R²_Y·TQ
