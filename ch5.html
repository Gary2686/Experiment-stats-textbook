正交多元迴歸
Orthogonal Multiple Regression

§1

為什麼要學「多元」迴歸？

第四章我們學會了用「一個自變數 X」預測「一個依變數 Y」。但現實世界中，影響一個結果的因素通常不只一個。

例如：

📚

影響考試成績的因素

不只讀書時間，還有 IQ、睡眠品質、壓力、做題量⋯⋯

💼

影響工作滿意度的因素

不只薪水，還有人際、自主性、成長性、與主管關係⋯⋯

🧠

影響記憶表現的因素

不只學習次數，還有干擾刺激、年齡、注意力、情緒⋯⋯

💡 多元迴歸（Multiple Regression）的核心思想
把「一個自變數 X」擴充為「多個自變數 X、T、…」，同時預測依變數 Y。本章先處理最簡單也最優雅的情況：兩個彼此「正交（orthogonal）」的自變數。

本章的核心是：

📖 一個自變數 → 迴歸線（2D 空間）
📐 兩個自變數 → 迴歸平面（3D 空間）
🌐 K 個自變數 → 迴歸超平面（K+1 維空間）

§2

從「線」到「平面」

第四章我們只有一個自變數 X，所以每筆資料是「一對數字」(X, Y)——可以畫在一張平面座標紙上（X 當橫軸、Y 當縱軸），最佳預測就是一條線。

現在多了第二個自變數 T，每筆資料變成「三個數字」(X, T, Y)。三個數字＝三個座標軸，資料點就飛離紙面、住進 3D 空間了。這時候若還硬用一條線去穿它，那條線只能朝一個方向斜，等於忽略掉另一個自變數。所以我們需要一個能同時朝兩個方向傾斜的東西——一片平面。

🧭 一句話抓住重點：
· 1 個自變數 → 資料是 2D 的點 → 用線預測（線只能往一個方向斜）
· 2 個自變數 → 資料是 3D 的點 → 用平面預測（平面能往兩個方向斜）
所以「多加一個自變數」＝「多給預測一個可以傾斜的方向」。這就是為什麼要升級成平面，而不是繼續用線。

🏔️ 生活比喻：把迴歸平面想成一片「斜坡」

想像腳下的地板是一張大地圖：往東走代表學習次數 X 變多，往北走代表干擾數 T 變多。你站在地圖上任何一點，抬頭看到的高度，就是模型「預測的回憶字數 Ŷ」。

把所有位置的高度連起來，就形成一片傾斜但平整的板子（沒有彎折，只是斜放）：

· 往東（X↑）走，地面越走越高 → 學得多、記得多（X 方向斜率 b > 0）
· 往北（T↑）走，地面越走越低 → 干擾多、記得少（T 方向斜率 c < 0）
· 你站在原點 (X=0, T=0) 的高度，就是截距 a

📐 迴歸平面長這樣：地板是 (X, T)，離地高度是預測值 Ŷ
藍色斜板就是迴歸平面；板上每一點的高度＝該 (X, T) 組合的預測回憶字數 Ŷ。
紅點是某位受試者的實際字數；他到平面那段紅色垂直線就是「殘差」——模型猜錯的量（§5、§9 會用到）。

用方程式寫出來，其實只是把第四章的直線「多接一段」而已：

第四章：直線（1 個自變數）
Ŷ = a + bX
a 起點高度　·　b X 方向的斜率
＋ 加上「第二個傾斜方向」↓
本章：平面（2 個自變數）
Ŷ = a + bX + cT
a = 截距：站在原點 (X=0, T=0) 的高度
b = X 方向斜率：T 固定不動時，X 每 +1，Ŷ 增加多少
c = T 方向斜率：X 固定不動時，T 每 +1，Ŷ 增加多少
↳ 唯一的新東西就是黃色那一項 +cT，也就是「平面的第二個傾斜方向」。
🎛️ 互動演示：調整 a、b、c，觀察平面形狀變化（俯視等高線圖）
目前方程式：Ŷ = 2 + 2X + 0.5T
等高線越密 → 該方向斜率越大 · 顏色代表 Ŷ 值（亮黃=高，深藍=低）
🎨 怎麼讀這張互動圖？它是把上面那片斜坡「從正上方俯視」
從正上方往下看平面，就像看登山地形圖：每條線串起「高度 Ŷ 相同」的位置。
· 顏色：亮黃＝Ŷ 高（記得多），深藍＝Ŷ 低（記得少）
· 等高線越密＝那個方向斜得越陡＝該自變數影響越大
· 動手試試：把 b 調大，左右方向的顏色會變化得更快（X 影響變強）；把 c 調成負的，上下方向的高低就會反過來。
§3

什麼是「正交」？為什麼它這麼重要？

「正交（Orthogonal）」是這一整章的關鍵詞。它的意思很簡單：

正交的定義
兩個自變數正交 ⟺ 它們的相關係數 r = 0
換句話說，兩變數彼此「無關」、「獨立變化」。
正交 ⟺ SCPXT = 0 ⟺ rXT = 0
⊥

正交 = 完全不相關

X 的變化不會「影響」或「預測」T 的變化。例如實驗中故意設計成：

· X = 學習次數（2、4、8）
· T = 干擾項數（2、4、8）
· 每個 (X, T) 組合都安排相同數量的受試者

🔗

非正交 = 有相關

X 和 T 之間有相關（r ≠ 0），會造成「共線性」問題：

· 例如：「讀書時間」和「練習題數」通常正相關
· 計算與解釋會變得很複雜
· 這是第六章的主題

🔬 為什麼研究者偏好正交設計？
1. 無混淆：自變數之間沒有重疊，可清楚分離各自效果
2. 計算簡單：公式可大幅簡化（後面會看到）
3. 解釋直觀：各變數的貢獻可以「加總」起來
4. 實驗設計：研究者可以「主動」操控變數使其正交（第 15 章）
§4

實例：倒攝抑制實驗（Slamecka, 1960）

讓我們用一個經典心理學實驗來貫穿整章。

🧠 什麼是「倒攝抑制」（Retroactive Interference）？

「後來學的東西，會讓我們忘掉之前學的東西」。

例如：先背了一張單字表，再背另一張，會發現第一張記得的單字變少了——這就是倒攝抑制。

實驗設計：

變數符號水準
自變數 1：原始學習次數X2 次 / 4 次 / 8 次
自變數 2：插入學習的清單數T2 個 / 4 個 / 8 個
依變數：正確回憶的字數Y連續數值

研究者預測：

📈 X（學習次數）越多 → Y（回憶字數）應該越多（正相關）
📉 T（干擾清單）越多 → Y（回憶字數）應該越少（負相關）

每個 (X, T) 組合各有 2 位受試者，總共 9 種組合 × 2 = 18 位受試者：

學習次數 X ↓ \ 干擾數 T →T = 2T = 4T = 8
X = 235, 3921, 316, 8
X = 440, 5234, 4218, 26
X = 861, 7358, 6646, 52
📊 視覺化：3 條線各代表一個 X 水準
X=2（紅）│ X=4（橘）│ X=8（綠）　·　每組顯示「平均」回憶字數
👀 看圖直觀感受：
· 三條線都往下傾斜 → 干擾越多 → 回憶越少（負相關）
· 線之間有明顯間距，且 X=8 最高、X=2 最低 → 學習次數越多 → 回憶越多（正相關）
· 三條線大致平行 → 兩個自變數的效果是「獨立」加總的（沒有交互作用，因為設計成正交）
§5

計算迴歸平面的 a、b、c

好幾片平面都能穿過這團 3D 資料，哪一片才「最佳」？跟第四章的直線一樣，用最小平方法來挑。

🎯 「最佳平面」是什麼意思？

回到 §2 的紅點：每個點（實際字數）到藍色平面的那段垂直距離，就是殘差 e ＝ 實際 − 預測（平面在這個人身上猜錯多少）。

把每個殘差平方後全部加起來，得到一個「總誤差」。能讓這個總誤差變到最小的那片平面，就是最佳平面。

為什麼要平方？①讓正、負誤差不會互相抵消；②對「錯很大」的點懲罰更重，逼平面盡量貼近每個人。

把「總誤差最小」這個條件用數學解開，就得到 a、b、c 的公式。公式有點長，但別怕——我們先看它在講什麼故事，再看它怎麼一秒變簡單。

🕵️ 為什麼 b 的公式這麼長？因為要先「洗掉 T 的影響」

一般情況下，X 和 T 多少會有點相關。若直接算 X 對 Y 的關係，裡面會混進一部分其實是 T 造成的效果，讓 X 「看起來」比實際更有用。
公式中被減掉的那一項 SCPXT·SCPTY，就是在「把混進來的 T 影響扣乾淨」，只留下 X 對 Y 的純效果。
X 方向的斜率 b（先看故事，別急著背）
b = (SST·SCPXY − SCPXT·SCPTY) / (SSX·SST − SCPXT²)
把長公式拆成幾塊看：
· SST·SCPXY：X 和 Y 的原始關聯
· − SCPXT·SCPTY：扣掉「透過 T 混進來」的部分（洗乾淨）
· 分母：把單位標準化

關鍵一秒簡化：本章 X 與 T 正交，SCPXT = 0，橘色那項整個消失、分母也少一塊，公式縮回第四章的樣子：
b = SST·SCPXY / (SSX·SST) = SCPXY / SSX（跟第四章一模一樣！）
T 方向的斜率 c（同樣的故事，角色對調）
c = (SSX·SCPTY − SCPXT·SCPXY) / (SSX·SST − SCPXT²)
一樣是「先把 X 的影響扣掉，再算 T 對 Y 的純效果」。
正交時橘色項＝0，簡化為：c = SCPTY / SST
🛒 用推車比喻理解「正交為何能簡化」：兩個人從不同角度推一台購物車。若兩人方向剛好垂直（正交），各推各的、互不干擾，你可以分開量每個人的貢獻——這正是為什麼正交時 b、c 各自用第四章的簡單公式就能算。若兩人方向有重疊（非正交），就得先扣掉「共同出力」的部分，公式才會變長——那是第六章的故事。
截距 a
a = MY − b·MX − c·MT
迴歸平面必定通過點 (MX, MT, MY)

🧮 套用到我們的範例

從原始資料計算這些基本統計量（細節省略，重點是看怎麼用）：

統計量意義數值
MXX 的平均4.67
MTT 的平均4.67
MYY 的平均39.33
SSXX 的平方和112
SSTT 的平方和112
SSYY 的平方和6,214
SCPXYX 和 Y 的交叉乘積和672
SCPTYT 和 Y 的交叉乘積和−448
SCPXTX 和 T 的交叉乘積和0 ← 正交！
✨ 神奇的時刻：SCPXT = 0
這代表 X 和 T 在實驗設計中完全正交。看下面公式怎麼瞬間簡化：
1

計算 b（X 的斜率）

原本公式：b = (SST·SCPXY − SCPXT·SCPTY) / (SSX·SST − SCPXT²)

因為 SCPXT = 0，所以：b = (112 × 672 − 0) / (112 × 112 − 0) = 75,264 / 12,544 = 6

意義：學習次數每增加 1 次，平均多記 6 個字。

2

計算 c（T 的斜率）

同樣地：c = (112 × (−448) − 0) / (112 × 112 − 0) = −50,176 / 12,544 = −4

意義：干擾清單每增加 1 個，平均忘掉 4 個字。

3

計算 a（截距）

a = MY − b·MX − c·MT
  = 39.33 − 6 × 4.67 − (−4) × 4.67
  = 39.33 − 28 + 18.67 = 30

4

寫出迴歸平面方程式

Ŷ = 30 + 6X − 4T

用文字描述：
「回憶字數 = 30 + 6 × 學習次數 − 4 × 干擾清單數」

🎯 方程式的實用性：可以做預測
假設一個受試者學了 5 次，被插入 3 個干擾清單，預測他能記得幾個字？
Ŷ = 30 + 6 × 5 − 4 × 3 = 30 + 30 − 12 = 48 個字
§6

預測得多好？R² 評估

我們算出了一條方程式，但它「預測得有多準」？跟第四章一樣，用 決定係數 衡量。但因為現在是多個自變數，要用大寫 R²：

多元決定係數 Multiple R²
R²Y·XT = SCPŶY² / (SSŶ · SSY)
讀作「R 平方，Y 被 X 和 T 解釋的部分」
本例：R² = 5,824² / (5,824 × 6,214) = 0.9372
🎯 解讀：兩個自變數（學習次數 + 干擾數）合起來解釋了 93.72% 的回憶字數變異！只剩 6.28% 無法解釋（誤差）。這個預測模型非常準。

F 檢定：這個 R² 是真的還是偶然？

多元迴歸的 F 比值
F = R² / (1 − R²) × (S − K − 1) / K
S = 觀察值數量（本例 18），K = 自變數數量（本例 2）
ν1 = K = 2，ν2 = S − K − 1 = 15
1

套入數值

F = 0.9372 / (1 − 0.9372) × (18 − 2 − 1) / 2
   = 0.9372 / 0.0628 × 15 / 2
   = 14.92 × 7.5 = 111.93

2

查 F 表

ν1=2, ν2=15, α=.05 → F臨界 = 3.68

3

下結論

F = 111.93 ≫ 3.68 → 拒絕 H₀
結論：兩個自變數合起來真的能線性預測 Y，p < .05

§7

每個自變數各自貢獻多少？

R² = 0.9372 是「兩個自變數合起來」的解釋力。但我們想知道：

📌 X（學習次數）單獨貢獻多少？
📌 T（干擾數）單獨貢獻多少？

分別計算每個 IV 與 Y 的相關係數平方：

X 對 Y 的解釋力
r²Y·X = SCPXY² / (SSX · SSY)
本例：r²Y·X = 672² / (112 × 6,214) = 451,584 / 695,968 = 0.6489　（X 單獨解釋了 64.89%）
T 對 Y 的解釋力
r²Y·T = SCPTY² / (SST · SSY)
本例：r²Y·T = (−448)² / (112 × 6,214) = 200,704 / 695,968 = 0.2884　（T 單獨解釋了 28.84%）

§8

✨ 正交的魔力：變異可加性

請仔細看下面這個結果——這是整章最美的數學發現：

🪄 正交時的變異可加性
r²Y·X + r²Y·T = R²Y·XT
0.6489 + 0.2884 = 0.9373 ≈ 0.9372 ✓
（差 0.0001 純粹是四捨五入造成）
🤔 為什麼這件事很神奇？

一般直覺：兩個自變數合起來的效果，應該不會剛好等於它們各自效果的總和——除非它們完全不重疊。

但這個結果告訴我們：當 X 和 T 正交時，它們對 Y 的解釋是「無重疊的」，所以可以直接相加。

這就像把蛋糕完美切成兩塊：每一塊都不會「跨界」到另一塊。
⚠️ 注意：這個「可加性」只在正交時成立！
如果 X 和 T 之間有相關（非正交），它們對 Y 的解釋會「重疊」，r²Y·X + r²Y·T 通常會大於 R²Y·XT（因為重疊部分被算了兩次）。這就是第六章「非正交多元迴歸」要處理的問題。

📐 為什麼會這樣？背後的數學

SSregression 的展開式中有一項：

SS_regression 的完整展開
SSreg = b²·SSX + c²·SST + 2bc·SCPXT
最後一項是「交叉項」，代表 X 和 T 共同解釋的部分。

當 X 和 T 正交時，SCPXT = 0，所以交叉項 = 0，公式簡化為：

SSreg = b²·SSX + c²·SST

這就是為什麼變異可以乾淨地分割成兩部分。

自由度怎麼來？

dfregression = K = 2：要決定一個平面需要 3 個點，扣掉均值點 1 個 = 2 個自由度

dfresidual = S − K − 1 = 18 − 2 − 1 = 15：每估一個參數（a, b, c）消耗 1 個自由度

§10

迴歸的「小字條款」：誤差項的三大假設

前面我們算出 Ŷ = 30 + 6X − 4T，並得到 SSresidual = 390。這個「殘差（residual）」，就是每位受試者的實際值 − 預測值——也就是模型「猜錯」的那一小塊 e = Y − Ŷ。

但迴歸要能正確地做檢定與預測，其實背後悄悄簽了一份「合約」：這些誤差（殘差）必須滿足三大假設。一旦違反，前面算出的 F 值、p 值、信賴區間就可能「看起來很漂亮，其實不能信」。

📜 把文言文翻成白話：迴歸不要求「資料」長得很完美，它只要求「模型猜錯的那部分（殘差）」乖乖聽話——要像一團沒有規律的白噪音：形狀常態（鐘形）、彼此不通風報信（獨立）、大小到處都差不多（等變異）。這三句話，就是下面三個假設。
🔔

① 常態性 Normality

白話：殘差的分佈要像鐘形——大多數靠近 0、極端值很少、左右對稱。（母體若常態，誤差項也會呈常態。）

怎麼看：常態機率圖（normal probability plot / Q-Q 圖），點沿對角線＝過關；或用 Shapiro-Wilk 檢定，p > .05 代表「不能拒絕常態」。

🔗

② 獨立性 Independency

白話：殘差之間要互不相關——前一個猜錯，不該「洩漏」下一個會怎麼錯。否則估計迴歸參數時會降低統計檢定力。

怎麼看：Durbin-Watson 檢定。DW ≈ 2＝獨立；DW → 0＝正自相關；DW → 4＝負自相關。

📢

③ 變異數同質性 Constant Variance

白話：不論預測值高或低，殘差的「散開程度」要一樣。若變異數不相等，自變數就無法有效估計依變數。

怎麼看：殘差圖（Residual Plot），把殘差對 ŷ 畫散佈圖——若呈喇叭 / 扇形就是違反（heteroscedasticity）。

1

跑迴歸、存下殘差

2

畫殘差圖（e vs ŷ）→ 找喇叭形

雲帶等寬＝變異數同質；越往右越發散（或收斂）＝違反。

3

畫常態機率圖 / 跑 Shapiro-Wilk → 看是否貼直線

點貼對角線＝常態；兩端翹起、整體彎曲＝偏態或厚尾。

4

有時間 / 空間順序時：畫序列圖 + 算 Durbin-Watson

殘差上下亂跳＝獨立；連續同號的波浪、DW 明顯偏離 2＝自相關。

5

有違反就對症下藥

轉換變數（如取對數）、補進遺漏的自變數、改用穩健標準誤，或換成更合適的模型。

⚠️ 一句話總結：假設不是「資料要漂亮」，而是「殘差要像白噪音」。三張診斷圖分工不同——殘差圖抓變異數、Q-Q 圖抓常態、序列圖＋DW抓獨立。任一違反，模型仍能算出數字，但那些數字（尤其 p 值與信賴區間）會誤導你。
🔗 延伸：這三大假設不只屬於迴歸——第 11 章「ANOVA 的基本假設」會再遇到同樣的常態性、變異數同質性、獨立性。因為 ANOVA 與迴歸其實是同一個「一般線性模型」的兩種面貌（第 9 章）。
§11

「獨立」其實有兩種：誤差要獨立，自變數也要獨立

很多人以為迴歸只要求「誤差獨立」，其實迴歸裡的「獨立」有兩個不同的要求，常被搞混。把它們分清楚，你就能看懂 §3「正交」到底在追求什麼，也能預告第六章的麻煩從哪來。

🔗

① 誤差項之間要獨立

要求：不同觀察值的殘差不能有「自我相關（autocorrelation）」——前一個猜錯，不該預告下一個怎麼錯。

檢查：Durbin-Watson（≈2 佳）。違反：檢定力下降、p 值失真。（§10 已示範過。）

🧲

② 自變數之間要獨立

要求：自變數 X、T 彼此不能高度相關，否則叫「多重共線性（multicollinearity）」。

檢查：VIF（變異數膨脹因子，>10 通常代表嚴重）。違反：標準誤膨脹、係數不穩、分不清是誰的功勞。

🔎 這一章的「正交」＝把 ② 做到完美：
正交就是 rXT = 0，兩個自變數完全不相關——這是「自變數獨立」的最理想狀態，此時 VIF = 1。反過來，若 rXT 越接近 1，共線性越嚴重，就是第六章要收拾的爛攤子。下面就來親手滑過這條光譜。

📝 考試申論題怎麼答？（Gauss-Markov 古典假設）

若申論題問「與自變數 X 有關的迴歸假設」，評分標準通常對應 Gauss-Markov 定理。直接列出這三大點最貼近標準答案：
1

參數線性（Linearity in Parameters）

母體模型中，Y 與 X 之間是參數（斜率係數）的線性組合，例如 Y = β₀ + β₁X₁ + β₂X₂ + ε。
關鍵字：參數線性、模型設定正確。

2

外生性：自變數與誤差項不相關（Exogeneity）

E(ε | X₁,…,X_k) = 0，即 Cov(X, ε) = 0。
關鍵字：外生性、無內生性（Endogeneity）。違反 → 係數偏誤。

3

無完全共線性（No Perfect Collinearity）

自變數彼此之間不能有完全線性關係——任何一個都不能被其他自變數線性組合完全取代（設計矩陣需滿秩）。
關鍵字：無多重共線性、Design Matrix 滿秩。

⚠️ 兩個常見的加分/陷阱補充：
· 外生性若違反（自變數與誤差相關）＝內生性問題，係數不再不偏。
· 若自變數本身帶有測量誤差，會造成「衰減偏差」——係數被往 0 的方向拉。
（常態性、變異數同質性、誤差獨立則屬於對「誤差項」的假設，見 §10——它們保證推論有效與 OLS 最有效率 BLUE，但通常不算在「與自變數有關」那一組。）

本章指定閱讀：地位的顏色

📄 Zhou, Xiao, Yoon, & Zhu (2026). The Color of Status: Color Saturation, Brand Heritage, and Perceived Status of Luxury Brands.

Journal of Consumer Research, 52(6), 1232–1249.（Open Access）

摘要：奢侈品用「低飽和度」的顏色（灰濛濛的「安靜奢華」色系），會讓消費者覺得品牌地位更高——因為褪色感讓人聯想到歲月，進而覺得品牌「有悠久的延續性傳承」。作者用 7 個研究層層檢驗：從 9 種飽和度的劑量反應（研究1）、只對奢侈品牌有效的交互作用（研究2）、傳承感的中介（研究3）、兩個邊界條件——強調近期創立年份時失效（研究4）、主打創新時反轉（研究5）——一路做到精品店的真實出價（+24.5% WTP，研究6）與自製手遊中的真實選擇（研究7）。

為什麼選這篇配第五章？它的 2×2／2×3 實驗全是完全交叉＋隨機分派的正交設計——主效果與交互作用得以乾淨分割，正是本章 §3、§8 的實戰版。它同時是一部「實驗統計方法目錄」：趨勢分析、多因子 ANOVA、中介與調節中介、BDM 出價、McNemar 檢定通通登場。
