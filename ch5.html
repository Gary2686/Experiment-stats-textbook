正交多元迴歸
Orthogonal Multiple Regression

當有兩個自變數時，迴歸線變成迴歸平面。本章揭示「正交」這個性質如何讓計算變得無比優雅。

§1

為什麼要學「多元」迴歸？

第四章我們學會了用「一個自變數 X」預測「一個依變數 Y」。但現實世界中，影響一個結果的因素通常不只一個。

例如：

📚

影響考試成績的因素

不只讀書時間，還有 IQ、睡眠品質、壓力、做題量⋯⋯

💼

影響工作滿意度的因素

不只薪水，還有人際、自主性、成長性、與主管關係⋯⋯

🧠

影響記憶表現的因素

不只學習次數，還有干擾刺激、年齡、注意力、情緒⋯⋯

💡 多元迴歸（Multiple Regression）的核心思想
把「一個自變數 X」擴充為「多個自變數 X、T、…」，同時預測依變數 Y。本章先處理最簡單也最優雅的情況：兩個彼此「正交（orthogonal）」的自變數。

本章的核心是：

📖 一個自變數 → 迴歸線（2D 空間）
📐 兩個自變數 → 迴歸平面（3D 空間）
🌐 K 個自變數 → 迴歸超平面（K+1 維空間）

§2

從「線」到「平面」

還記得第四章的直線方程式嗎？

直線方程式（單變數）
Ŷ = a + bX
a = 截距（X=0 時 Y 的值）
b = 斜率（X 每變 1 單位，Y 變化的量）

現在我們有兩個自變數 X 和 T。只要把「斜率項」多加一個就好：

平面方程式（兩個自變數）
Ŷ = a + bX + cT
a = 截距（當 X=0、T=0 時 Y 的值）
b = X 方向的斜率（T 固定時，X 每變 1 單位，Y 變化的量）
c = T 方向的斜率（X 固定時，T 每變 1 單位，Y 變化的量）
🎛️ 互動演示：調整 a、b、c，觀察平面形狀變化（俯視等高線圖）
目前方程式：Ŷ = 2 + 2X + 0.5T
等高線越密 → 該方向斜率越大 · 顏色代表 Ŷ 值（亮黃=高，深藍=低）
🎨 怎麼讀這個圖？
這是平面的「俯視等高線圖」，就像地形圖一樣。每條線代表 Ŷ 值相同的位置。
· 線的密度：越密代表該方向斜率越大、Ŷ 變化越快
· 線的方向：垂直於線的方向就是「上坡」最快的方向
· 顏色深淺：直接代表 Ŷ 的大小
§3

什麼是「正交」？為什麼它這麼重要？

「正交（Orthogonal）」是這一整章的關鍵詞。它的意思很簡單：

正交的定義
兩個自變數正交 ⟺ 它們的相關係數 r = 0
換句話說，兩變數彼此「無關」、「獨立變化」。
正交 ⟺ SCPXT = 0 ⟺ rXT = 0
⊥

正交 = 完全不相關

X 的變化不會「影響」或「預測」T 的變化。例如實驗中故意設計成：

· X = 學習次數（2、4、8）
· T = 干擾項數（2、4、8）
· 每個 (X, T) 組合都安排相同數量的受試者

🔗

非正交 = 有相關

X 和 T 之間有相關（r ≠ 0），會造成「共線性」問題：

· 例如：「讀書時間」和「練習題數」通常正相關
· 計算與解釋會變得很複雜
· 這是第六章的主題

🔬 為什麼研究者偏好正交設計？
1. 無混淆：自變數之間沒有重疊，可清楚分離各自效果
2. 計算簡單：公式可大幅簡化（後面會看到）
3. 解釋直觀：各變數的貢獻可以「加總」起來
4. 實驗設計：研究者可以「主動」操控變數使其正交（第 15 章）
§4

實例：倒攝抑制實驗（Slamecka, 1960）

讓我們用一個經典心理學實驗來貫穿整章。

🧠 什麼是「倒攝抑制」（Retroactive Interference）？

「後來學的東西，會讓我們忘掉之前學的東西」。

例如：先背了一張單字表，再背另一張，會發現第一張記得的單字變少了——這就是倒攝抑制。

實驗設計：

變數符號水準
自變數 1：原始學習次數X2 次 / 4 次 / 8 次
自變數 2：插入學習的清單數T2 個 / 4 個 / 8 個
依變數：正確回憶的字數Y連續數值

研究者預測：

📈 X（學習次數）越多 → Y（回憶字數）應該越多（正相關）
📉 T（干擾清單）越多 → Y（回憶字數）應該越少（負相關）

每個 (X, T) 組合各有 2 位受試者，總共 9 種組合 × 2 = 18 位受試者：

學習次數 X ↓ \ 干擾數 T →T = 2T = 4T = 8
X = 235, 3921, 316, 8
X = 440, 5234, 4218, 26
X = 861, 7358, 6646, 52
📊 視覺化：3 條線各代表一個 X 水準
X=2（紅）│ X=4（橘）│ X=8（綠）　·　每組顯示「平均」回憶字數
👀 看圖直觀感受：
· 三條線都往下傾斜 → 干擾越多 → 回憶越少（負相關）
· 線之間有明顯間距，且 X=8 最高、X=2 最低 → 學習次數越多 → 回憶越多（正相關）
· 三條線大致平行 → 兩個自變數的效果是「獨立」加總的（沒有交互作用，因為設計成正交）
§5

計算迴歸平面的 a、b、c

跟第四章一樣，最佳平面就是讓「所有點到平面的垂直距離平方總和最小」（最小平方法）。

結果可以用以下公式計算：

X 方向的斜率 b（看起來複雜，但別怕）
b = (SST·SCPXY − SCPXT·SCPTY) / (SSX·SST − SCPXT²)
這個複雜公式是「把 T 的影響先扣掉，再算 X 對 Y 的純效果」。
但是！當 X 和 T 正交時，SCPXT = 0，公式立刻簡化為：
b = SST·SCPXY / (SSX·SST) = SCPXY / SSX（這跟第四章一模一樣！）
T 方向的斜率 c
c = (SSX·SCPTY − SCPXT·SCPXY) / (SSX·SST − SCPXT²)
正交時簡化為：c = SCPTY / SST
截距 a
a = MY − b·MX − c·MT
迴歸平面必定通過點 (MX, MT, MY)

🧮 套用到我們的範例

從原始資料計算這些基本統計量（細節省略，重點是看怎麼用）：

統計量意義數值
MXX 的平均4.67
MTT 的平均4.67
MYY 的平均39.33
SSXX 的平方和112
SSTT 的平方和112
SSYY 的平方和6,214
SCPXYX 和 Y 的交叉乘積和672
SCPTYT 和 Y 的交叉乘積和−448
SCPXTX 和 T 的交叉乘積和0 ← 正交！
✨ 神奇的時刻：SCPXT = 0
這代表 X 和 T 在實驗設計中完全正交。看下面公式怎麼瞬間簡化：
1

計算 b（X 的斜率）

原本公式：b = (SST·SCPXY − SCPXT·SCPTY) / (SSX·SST − SCPXT²)

因為 SCPXT = 0，所以：b = (112 × 672 − 0) / (112 × 112 − 0) = 75,264 / 12,544 = 6

意義：學習次數每增加 1 次，平均多記 6 個字。

2

計算 c（T 的斜率）

同樣地：c = (112 × (−448) − 0) / (112 × 112 − 0) = −50,176 / 12,544 = −4

意義：干擾清單每增加 1 個，平均忘掉 4 個字。

3

計算 a（截距）

a = MY − b·MX − c·MT
  = 39.33 − 6 × 4.67 − (−4) × 4.67
  = 39.33 − 28 + 18.67 = 30

4

寫出迴歸平面方程式

Ŷ = 30 + 6X − 4T

用文字描述：
「回憶字數 = 30 + 6 × 學習次數 − 4 × 干擾清單數」

🎯 方程式的實用性：可以做預測
假設一個受試者學了 5 次，被插入 3 個干擾清單，預測他能記得幾個字？
Ŷ = 30 + 6 × 5 − 4 × 3 = 30 + 30 − 12 = 48 個字
§6

預測得多好？R² 評估

我們算出了一條方程式，但它「預測得有多準」？跟第四章一樣，用 決定係數 衡量。但因為現在是多個自變數，要用大寫 R²：

多元決定係數 Multiple R²
R²Y·XT = SCPŶY² / (SSŶ · SSY)
讀作「R 平方，Y 被 X 和 T 解釋的部分」
本例：R² = 5,824² / (5,824 × 6,214) = 0.9372
🎯 解讀：兩個自變數（學習次數 + 干擾數）合起來解釋了 93.72% 的回憶字數變異！只剩 6.28% 無法解釋（誤差）。這個預測模型非常準。

F 檢定：這個 R² 是真的還是偶然？

多元迴歸的 F 比值
F = R² / (1 − R²) × (S − K − 1) / K
S = 觀察值數量（本例 18），K = 自變數數量（本例 2）
ν1 = K = 2，ν2 = S − K − 1 = 15
1

套入數值

F = 0.9372 / (1 − 0.9372) × (18 − 2 − 1) / 2
   = 0.9372 / 0.0628 × 15 / 2
   = 14.92 × 7.5 = 111.93

2

查 F 表

ν1=2, ν2=15, α=.05 → F臨界 = 3.68

3

下結論

F = 111.93 ≫ 3.68 → 拒絕 H₀
結論：兩個自變數合起來真的能線性預測 Y，p < .05

§7

每個自變數各自貢獻多少？

R² = 0.9372 是「兩個自變數合起來」的解釋力。但我們想知道：

📌 X（學習次數）單獨貢獻多少？
📌 T（干擾數）單獨貢獻多少？

分別計算每個 IV 與 Y 的相關係數平方：

X 對 Y 的解釋力
r²Y·X = SCPXY² / (SSX · SSY)
本例：r²Y·X = 672² / (112 × 6,214) = 451,584 / 695,968 = 0.6489　（X 單獨解釋了 64.89%）
T 對 Y 的解釋力
r²Y·T = SCPTY² / (SST · SSY)
本例：r²Y·T = (−448)² / (112 × 6,214) = 200,704 / 695,968 = 0.2884　（T 單獨解釋了 28.84%）

📊 視覺化：每個自變數的貢獻

學習次數 X 解釋的變異r²Y·X = 64.89%
X (64.89%)
干擾數 T 解釋的變異r²Y·T = 28.84%
T (28.84%)
兩者合計 R²Y·XT= 93.72% ✓

§8

✨ 正交的魔力：變異可加性

請仔細看下面這個結果——這是整章最美的數學發現：

🪄 正交時的變異可加性
r²Y·X + r²Y·T = R²Y·XT
0.6489 + 0.2884 = 0.9373 ≈ 0.9372 ✓
（差 0.0001 純粹是四捨五入造成）
🤔 為什麼這件事很神奇？

一般直覺：兩個自變數合起來的效果，應該不會剛好等於它們各自效果的總和——除非它們完全不重疊。

但這個結果告訴我們：當 X 和 T 正交時，它們對 Y 的解釋是「無重疊的」，所以可以直接相加。

這就像把蛋糕完美切成兩塊：每一塊都不會「跨界」到另一塊。
⚠️ 注意：這個「可加性」只在正交時成立！
如果 X 和 T 之間有相關（非正交），它們對 Y 的解釋會「重疊」，r²Y·X + r²Y·T 通常會大於 R²Y·XT（因為重疊部分被算了兩次）。這就是第六章「非正交多元迴歸」要處理的問題。

📐 為什麼會這樣？背後的數學

SSregression 的展開式中有一項：

SS_regression 的完整展開
SSreg = b²·SSX + c²·SST + 2bc·SCPXT
最後一項是「交叉項」，代表 X 和 T 共同解釋的部分。

當 X 和 T 正交時，SCPXT = 0，所以交叉項 = 0，公式簡化為：

SSreg = b²·SSX + c²·SST

這就是為什麼變異可以乾淨地分割成兩部分。
§9

平方和分割（變異來源分析）

跟第四章一樣，可以把 Y 的總變異分解：

變異分割
SStotal = SSregression + SSresidual
本例：6,214 = 5,824 + 390
而 SSregression 又可進一步分成 X 和 T 各自的部分：
SSregression = SSY·X + SSY·T = 4,032 + 1,792 = 5,824 ✓
變異來源SS（平方和）df（自由度）MS（均方）F
迴歸總 (Regression)5,82422,912111.93
└ 由 X 解釋4,03214,032154.90
└ 由 T 解釋1,79211,79268.88
殘差 (Residual)3901526.0—
總計6,21417——
✅ 結論：
· 整體模型顯著：F(2, 15) = 111.93, p < .05, R² = .94
· X（學習次數）顯著：F(1, 15) = 154.90, p < .05
· T（干擾數）顯著：F(1, 15) = 68.88, p < .05
· 兩個自變數各自都能顯著預測依變數
📐

自由度怎麼來？

dfregression = K = 2：要決定一個平面需要 3 個點，扣掉均值點 1 個 = 2 個自由度

dfresidual = S − K − 1 = 18 − 2 − 1 = 15：每估一個參數（a, b, c）消耗 1 個自由度
